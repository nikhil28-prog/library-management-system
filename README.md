# Library Management System (Python + SQLite)

This is a simple Library Management System built using Python and SQLite.
It allows users to manage books and student records efficiently.

## Features

* Add new book
* View all books
* Search book
* Update book details
* Delete book
* Database storage using SQLite

## Technologies Used

* Python
* SQLite
* VS Code
* Git & GitHub

## How to Run the Project

1. Install Python
2. Open project in VS Code
3. Run the program:

```bash
python library_management_system.py
```
